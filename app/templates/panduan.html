Panduan Aplikasi


1. Aplikasi ini adalah aplikasi yang dapat mengidentifikasi formalin pada ikan asin jenis peda

2. Cara menggunakan aplikasi adalah user masuk ke fitur aplikasi

3. Didalam fitur aplikasi user diharuskan memasukkan input berupa gambar ikan asin peda (harap sesuaikan dengan contoh)

4. Setelah user input gambar yang ingin diidentifikasi klik tombol prediksi, maka aplikasi akan mengidentifikasi ikan secara otomatis

5. Aplikasi akan menampilkan hasil prediksi berdasarkan input yang user masukkan
